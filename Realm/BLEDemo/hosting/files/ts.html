Timeseries data is normally a tuple of ( Time, Source, Reading(s))

For example:
  • Time, Stock, Price
  • Time, Vehicle, [ Speed , Fuel, Location]
Data in Database is recorded in Blocks (Because that's how Disks work)

Data is Read/Written and cached by the block.

New data fills empty blocks sequentially

Each data point has 2 index entries, an _id and its device and time.
